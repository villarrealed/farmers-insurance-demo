## Farmers Insurance landing page

- Built a self-contained `index.html` landing page to route users into the demo's `mobile.html` and `web.html` experiences.
- Visual direction: light gradient backdrop, glassmorphism shell, Farmers red accent system, subtle load/hover animation, and presentation-friendly card layout.

### Rebuild prompt

Create a self-contained `index.html` landing page for the Farmers Insurance demo that matches the branding in `web.html`, includes clickable cards for `mobile.html` and `web.html`, uses inline CSS only, keeps a polished glassmorphism aesthetic, and includes a footer noting it is powered by Cisco Webex Connect.

### Dependencies / references

- `web.html` for palette and overall atmosphere
- `mobile.html` for the mobile demo concept
- Brand colors: `#cc2228`, `#92191d`, `#fff1f1`, `#1f2230`, `#687086`

### Key decisions

- Used a text-based Farmers wordmark with a shield-inspired inline SVG badge rather than pulling in an external asset.
- Kept the landing page distinct from the portal while still visually tied to the existing demo.
